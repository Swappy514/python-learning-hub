# Functions

Functions group reusable code into named blocks.

- **Define** a function using `def` keyword.
- **Call** the function by its name.

## Example:

def greet(name):
print(f"Hello, {name}!")

greet("Alice")

def add(a, b):
return a + b

print(add(3, 4)) # Output: 7

Functions improve modularity and reduce code repetition.
